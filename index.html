Human Checkpoint

人工檢核 — 美感與創意，AI 無法取代

Agent 在每個重要時機點都會插入人工檢核，由實戰職能把關 — AI 跑流程，但最終的美感、創意、品味、系統架構由人決定

🔄 每個檢核點的 Loop 機制
🤖 Agent 產出
→
👀 實戰職能審核
→
✓ 通過 → 進入下一個 Agent
✗ 退回 → AI 依反饋修正 → 重新審核
🛡 為什麼要插？
AI 可以快、可以多，但美感與創意仍要由人定奪。每個重要時機點都安插檢核，攔截錯誤在源頭
🎯 檢核什麼？
編導 → 玩法創意與機制
美術 → 風格品味與一致性
音效 → 聽感與情境匹配
Client/Server → 邏輯正確性
機率 → 數值合理性
📈 產出什麼？
每次通過/退回自動紀錄，形成可量化的 Before/After 數據，給彙整推銷角色做簡報

🎯 為什麼要「實戰職能」？

🔄 我的角色 vs 你的角色

你（實戰職能）：在真實工作中使用、反饋問題、產出 Before/After
不需要會寫 AI、不需要學 prompt — 你只要把 Agent 當工具用，告訴我哪裡好用、哪裡不好用
